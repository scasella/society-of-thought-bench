# Society of Thought Demo

This Space lets you paste a prompt and inspect the raw paper-style `<think>` trace from the current preview checkpoint.

What it shows:

- a visible multi-persona debate inside the exposed thinking trace
- a separate final answer
- the exact raw model output that was returned

What it does not claim:

- that the model is fully reliable
- that the gain is proven outside this benchmark
- that this is a final release

Related links:

- Benchmark repo: https://github.com/scasella/society-of-thought-bench
- Adapter repo: https://huggingface.co/scasella91/society-of-thought-qwen3-30b-paper-faithful-adapter
