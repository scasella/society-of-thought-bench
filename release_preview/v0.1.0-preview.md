# v0.1.0-preview

This release publishes the first public preview of **Society of Thought**.

The release includes:

- a public adapter for `Qwen/Qwen3-30B-A3B`
- a live demo for inspecting the visible reasoning trace
- the benchmark and result files that support the released claim
- audit materials and helper scripts for inspection and evaluation

Headline result:

- debate average score: `0.732`
- monologue average score: `0.197`
- score gap: `+0.535`

Interpretation:

- the release centers on a visible multi-persona reasoning behavior
- the benchmark provides the evidence layer for that behavior
- the strongest evidence currently comes from the released medium comparison
- this is not yet a broader claim about transfer outside this setup
