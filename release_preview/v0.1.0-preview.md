# v0.1.0-preview

This release publishes the first public preview of `society-of-thought-bench`.

The release includes:

- the benchmark package
- the public adapter release for `Qwen/Qwen3-30B-A3B`
- the main debate-versus-monologue comparison
- a live demo and supporting release materials

Headline result:

- debate average score: `0.732`
- monologue average score: `0.197`
- score gap: `+0.535`

Also included:

- supporting result files for medium and hard runs
- additional validation and audit materials for the released checkpoint
- helper scripts for inspection, evaluation, and training

Scope:

- This is an experimental research preview.
- The strongest evidence comes from the released comparison within this benchmark.
- It is not yet a broader claim about transfer outside this setup.
